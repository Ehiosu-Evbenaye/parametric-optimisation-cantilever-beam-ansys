# Parametric Optimisation of a Steel Cantilever Beam Using ANSYS Workbench

*Finite Element Analysis & Design Optimisation | In progress (started 2026)*

Static structural analysis and parametric optimisation of an S355 steel cantilever beam in ANSYS Workbench. Preliminary work suggests a potential mass reduction of approximately 18% through taper optimisation while maintaining a safety factor of 2.0, pending full validation.

## Status
- Baseline model validated against Euler–Bernoulli beam theory (deflection error <3%).
- Mesh convergence study indicates stress variation <2% for refined meshes.
- Taper optimisation is ongoing; final results and full report will be added upon completion.
